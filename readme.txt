== Description ==

Fayzakseo Quick Contact is a powerful WordPress plugin that allows you to track real user clicks on phone and WhatsApp buttons – without registration, API keys, or third-party services.

The floating contact button is fully customizable:
- Move it to any side of the screen
- Change its color, position, and height
- Set your own phone and WhatsApp numbers

See a full list of who clicked and when – right in your WordPress dashboard.

Designed for site owners who care about real engagement and want to know if users are actually trying to contact them.

Perfect for small businesses, freelancers, landing pages, and eCommerce stores.

For a full explanation in Hebrew, visit: https://fayzakseo.com/fayzak-תוסף

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory or install via the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings page to set your contact numbers and appearance.

== Frequently Asked Questions ==

= Is it free to use? =
Yes, the plugin is 100% free and does not require any registration.

= Where can I see the click stats? =
After activation, go to the plugin's "Statistics" page in your WordPress admin.

= Can I reset the data? =
Yes, there's a protected admin-only reset option with a password.
